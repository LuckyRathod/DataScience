	** When to Use Which Test ?	
	- Considering One Categorical Feature - One Sample Proportion Test
	- Considering Two Categorical Features - Chi Square Test 
	- Considering One Continous Features - T test
	- Considering Two Continous Feature - Correlation - T test
	- Considering One Continous and One Categorical (<2 categories - T test)
	- Considering One Continous and One Categorical (>2 categories - Anova Test)
